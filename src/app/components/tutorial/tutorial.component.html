How is it calculated?

The Block Check Character is calculated by XOR-ing each byte with the result of previous XOR.

Example

In this example we will calculate the Block Check Character of the message
A4 37 F6 F8 CD.

Binary representations of each byte

A4: 1010 0100
37: 0011 0111
F6: 1111 0110
F8: 1111 1000
CD: 1100 1101

First XOR (Start with 0)

0000 0000      0
1010 0100     A4
---------
1010 0100     A4

Second XOR

1010 0100     A4
0011 0111     37
---------
1001 0011     93

Third XOR

1001 0011     93
1111 0110     F6
---------
0110 0101     65

Fourth XOR

0110 0101     65
1111 1000     F8
---------
1001 1101     9D

Fifth XOR

1001 1101     9D
1100 1101     CD
---------
0101 0000     50

Result

The result of the calculation will be the result of the last XOR
50
